### Thousand Tuning 千调工艺
千调工艺是本模组内的玩法之一，玩家需前往地下洞穴的**千调工枢**，开启**千调基点**以来解锁改内容。后续可以在各环境寻找**千调工站**以扩展千调工艺内容。
在现设计中，其有三种模块化工作方式，分别为**单体**，**系统**及**交叉化**
    
    单体：是指只依靠器械，无仪器和模块进行区域化工作的方式
    系统：是指在器械的基础上，装载仪器和模块，或只装载其中一种的，进行区域化工作的方式
    交叉化：是指单体和系统的联合利用进行的区域化工作的方式

关于模组内实际的物品类别，分别为**器械**，**仪器**，**模块**

    器械：有自身独立的功能，可以独立运作，可以安置插件
    仪器：有自身的功能，但无法独立运作，需安置在器械才能工作
    模块；主要用于扩展器械的功能，也可用于升级器械的功能。但有基础限制，作用单一

另外注意，千调基点解锁的是每个工艺的前期内容，后期的大部分内容还需要玩家自行解锁 *（具体怎么解锁和解锁什么等内容还没有设计捏）*

### 栽培工艺及环境区
栽培，顾名思义，其是关联作物，植物一类的种植及采收。但在栽培工艺内做了一些延伸

新设计的内容中存有作物一栏，其除对向**人工栽种**外，还对向于栽培工艺。在现设计的内容中，玩家可以进行栽种生产的有**植物**，**蘑菇**，**晶石**(这也是作物的类别)，
并且作物还分成三种培养方式，**土培**，**水培**，**半水培**，对于人工栽种来说，作物种植还有环境的限制。

**环境区**
隶属于栽培工艺内的一个分类，现设计中属于后期内容

类似于环境瓶建筑，其会在一定区域内生成一个不影响外界的环境区域，玩家可以自由进出

生成使用到的设备被称为**系统化液炼环境载析机**-简称**SL-Rec-A**。总共结合了**注炼机** *(器械)*，**易载式存储模块** *(模块)*，**载析仪** *(仪器)*三种物品

其拥有正常环境所具有的功能。会自动生产该环境内除液体，单次获得物外的任何物品，例如木材，敌怪掉落物。但在原本环境内一些物品产出受概率限制，因此一些受概率影响较大的物品（可以理解为玩家在环境内非必得的物品）在生产中会受时间的限制。对于生成环境，玩家需要制作对应环境的**微境注化剂**，使用于SL-Rec-A中便可生成对应环境区

有个小小的延伸项，还没有完全决定。关联于**非自然生成环境**

    非自然生成环境：定义非常简单，就是玩家在游玩时无法发现的环境，被称为非自然生成环境。
    它们通过自然生成环境延伸而来，其相关的微境注化剂通过自然生成环境里的物品而制作

在现设计中，只有一个属于该类别，其自然生成环境还未命名，其称为***酸腐林***。

### 机动工艺
本工艺是关于玩家移动类的一些内容，也会在后期做一些延伸

**滑索类**

    轻调滑索：使玩家在两点之间能够快速的运动
    重调滑索：使物品在区域内或两点间快速运输（请勿将其与两地之间的远程运输搞混）
    折式滑索：便携式可自主使用收纳并返回到玩家身边的滑索。就是两点之间一点缺失滑索时，玩家可以放置使用。
    使用完毕后，该设备自动折叠，设备内的无人机会收纳并返回于玩家。但需要定时充电才可长久使用

**-千调工艺现设计的内容只有这么多，后续还会继续设计-**

### 光学
光学是本模组内的玩法之一，与千调工艺类似，但有所差别。因为设计的时间比较短，内容还并不多

光学各个仪器被称为**晶仪**，顾名思义，其用晶石制作，依靠晶石工作。现设计的有五种晶仪，也分别对应光学内的五种状态

    光折镜-(单路)：向一点输送光能
    反折镜-(返回)：使光折镜输送光能时能够折返回来
    分折镜-(分路)：向至少三点输送光能
    凸折透镜-(聚集)：提升光能提供的功率，可用于“光路超载”
    凹折透镜-(分散)：向至少两点输送光能，但光能的含量会由1变为1/2，可用于“减率光路”
    
**-光学现设计的内容只有这么多，后续还会继续设计-**

#### 建筑地形

**千调工枢**

    生成于地下的占比区域比较大的一个建筑地形。它由多个舱室组成，每个舱室都有其设定的功能；
    对于流程而言，玩家可以在此处开启千调基点（前文提到过）以开始探索千调工艺玩法

**千调工站·密林温室**

    生成于丛林地下的一个建筑地形。它只有一个大型舱室，又称温室。外表呈半圆。
    里面种植着各种各样的作物，箱子内也存放着关于作物种子及栽种品等；
    对于流程而言，玩家可以开启此处的千调基点，以解锁基础-栽培工艺

**落花墓**

    生成于地表（具体位置还未决定，只是大概设想在森林环境）。它的大体结构可以分为外圈和内圈，外圈生成有娇白枫花（作物）及其延伸品，
    而内圈是一个拱门，但略有破损且有花枝缠绕。拱门下是一个墓碑（碑文还未设计）；
    对于流程而言，玩家可以利用灵龛内所获得的的道具，结识埃尔塔，完成埃尔塔的一些任务后，他会加入玩家队伍，变为旅伴。
    玩家会在此处遇见模组内的第一个Boss-“娇枫蝣”

**灵龛**

    同样生成于地下，大体结构还未设计；设计其的目的是为了方便玩家重新挑战本模组内的Boss，并加以强化的。因为本模组内的大部分Boss刚开始于挑战时
    是不借助于召唤物召唤的；当玩家来到此处时可以捡拾到一个能够观察到魂的道具。
    玩家击败模组内的第一个Boss时（现是娇枫蝣），可前往此处获得加强版的其的召唤物，往后亦是如此

#### NPC及分支

**NPC**

依照设计而顺序排列

    诺伊，桑克拉，艾凡，埃尔塔，汐卡莉

**旅伴及携力配队**：
旅伴是指加入玩家队伍的，可以释放战技来辅助玩家战斗的NPC，其包括:

    诺伊，桑克拉，艾凡，埃尔塔

**携力配队**：
它们分别是两个词，携力与配队

    携力：在模组内是指旅伴所释放的战技总称，其设计目的是辅助玩家进行战斗。拥有战技的除旅伴外，玩家自身也存在战技
    配队：在模组内，玩家可以在“旅伴携力”栏选择自己的旅伴。旅伴人员的上限数量为四，玩家可以自由地选择和调换。
    在线设计中，包括玩家自身在内的共有五个旅伴

值得注意的是，所谓战技和配队与其他游戏唯一不同的点在于，不用考虑什么属性，连谐之类的东西
