# CelestialOcean-Mod  v0.1<br>
## 沧海浮穹-模组
沧海浮穹是一个基于泰拉瑞亚而开发的模组，现阶段的设计路线包括但不限于新的玩法，机制，物品等。本模组主要以增添新内容为主，除了特殊情况外基本上不会对原本内容进行修改<br>
本模组主要以探索为主，因此不会考虑脱离设计项实际所在阶段的内容，并且本模组可能还会频繁的对原有内容进行修改，重置，删除。也正因此这些内容会搁置于的本模组附属包: **海之汐-Tide of the Sea**
